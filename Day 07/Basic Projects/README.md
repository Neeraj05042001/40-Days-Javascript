# 🚀 Beginner Level JavaScript Projects Practice 

This folder contains beginner-level JavaScript projects to  strengthen concepts like variables, data types, control flow, loops, and functions.  
Projects are listed in the order of increasing difficulty.

---

## 1️⃣ Even-Odd Number Checker

### 📋 Project Details:
- Take a number input from the user.
- Check whether the number is even or odd using the modulus `%` operator.
- Display the result.

### ✅ To-Do:
- Prompt user to enter a number.
- Write a function `isEven(number)` that returns true/false.
- Use `if-else` to check the returned value and display "Even" or "Odd".

---

## 2️⃣ Palindrome Checker

### 📋 Project Details:
- Check if a given word or number reads the same backward as forward (palindrome).

### ✅ To-Do:
- Prompt user to input a word or number.
- Write a function `isPalindrome(input)`:
  - Convert input to a string (if not already).
  - Reverse the string.
  - Compare original and reversed strings.
- Display whether it's a palindrome or not.

---

## 3️⃣ Simple Calculator

### 📋 Project Details:
- Create a simple calculator that can add, subtract, multiply, or divide two numbers.

### ✅ To-Do:
- Prompt the user for:
  - First number
  - Operation (`+`, `-`, `*`, `/`)
  - Second number
- Write a function `calculate(num1, operator, num2)` that returns the result.
- Use `if-else` or `switch-case` to perform the correct operation.
- Display the result.

---

## 4️⃣ Number Guessing Game

### 📋 Project Details:
- Build a game where the computer randomly picks a number, and the user tries to guess it.

### ✅ To-Do:
- Generate a random number between 1 and 100.
- Prompt the user to guess the number.
- Give feedback:
  - "Too High", "Too Low", or "Correct!"
- Keep track of the number of attempts.
- Allow replaying the game (optional).

---

## 5️⃣ Rock-Paper-Scissors Game

### 📋 Project Details:
- Build the classic Rock-Paper-Scissors game where the user plays against the computer.

### ✅ To-Do:
- Prompt the user to choose `rock`, `paper`, or `scissors`.
- Computer randomly selects one.
- Compare choices based on winning rules:
  - Rock beats Scissors
  - Scissors beats Paper
  - Paper beats Rock
- Display the winner (User wins, Computer wins, or Draw).
- Allow replaying the game (optional).

---

## 6️⃣ Countdown Timer

### 📋 Project Details:
- Create a countdown timer that starts from a user-given number and counts down to 0.

### ✅ To-Do:
- Prompt the user for a starting number.
- Use `setInterval()` to decrease the number by 1 every second.
- Log the countdown in the console.
- When it reaches 0, clear the interval and display "Time's Up!".

## 7️⃣ Random Quote Generator

### 📋 Project Details:
- Show a random motivational quote each time the user triggers an action.

### ✅ To-Do:
- Create an array of quotes.
- Write a function `getRandomQuote()`:
  - Pick a random quote from the array.
- Display the selected quote.

## 8️⃣ To-Do List (Console Version)

### 📋 Project Details:
- Build a simple to-do list app that runs in the console.

### ✅ To-Do:
- Create an empty array to store tasks.
- Provide options to:
  - Add a task

  - Delete a task

## 9️⃣ Simple Quiz App (Console)

### 📋 Project Details:
- Create a mini quiz application that asks multiple questions.

### ✅ To-Do:
- Create an array of questions and answers.
- Loop through each question:
  - Prompt the user for an answer.
  - Check if the answer is correct.
  - Keep a score.
